## 🎮 **Описание**  
Реализация классической игры **"Мафия"** на **React**, включающая:  
- Настройку ролей (мафия, шериф, доктор и др.)  
- Распределение ролей между игроками  

Игра подходит для быстрого старта вечеринки и обучения командной коммуникации.

---

## 🔧 **Основные функции**  
| Функция | Описание |
|--------|----------|
| **Настройка ролей** | Добавление, удаление, изменение количества ролей (например, 2 мафии, 5 мирных, 1 доктор). |
| **Ввод имён игроков** | Простая форма для заполнения имён перед началом игры. |
| **Распределение ролей** | Автоматическое распределение ролей между игроками с отображением результатов. |
| **Детализация ролей** | Описания и изображения для каждой роли (например, "Шериф может проверять игроков ночью"). |
