Основные функции
Настройка ролей:
Добавление, удаление, изменение количества ролей (например, 2 мафии, 5 мирных, 1 доктор).
Ввод имён игроков:
Простая форма для заполнения имён перед началом игры.
Распределение ролей:
Автоматическое распределение ролей между игроками с отображением результатов.
Детализация ролей:
Описания и изображения для каждой роли (например, "Шериф может проверять игроков ночью").
